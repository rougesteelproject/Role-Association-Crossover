The Role Assosiation Crossover is a database of actors and their roles. It links roles to actors and actors to roles in an ever-expanding web. The fun comes in that we treat each combination as one single character, combining powers, skills, and relationships. Use the (uppy-downy-arrow-things) at the top of each page to watch the web grow and the character increse in power, or use the Web Visualizer to see where the major connections lie. (Not ready yet)

Inspired by classics like Tenzin's Lemon Rant [Our Version], Sherlock Holmes as a Man of Iron [Our Version], and "These character have the same voice actors"
