What is the Role Association Crossover? How does it work?

The Role Assosiation Crossover is a self-updating database of actors and their roles. It links roles to actors and actors to roles in an ever-expanding web. The fun comes in that we treat each combination as one single character, combining powers, skills, and relationships. Use the slider at the top of each page to watch the web grow and the character increse in power, or use the Web Visualizer to see where the major connections lie.
